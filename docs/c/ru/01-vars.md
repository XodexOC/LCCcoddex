# Урок 01: Переменные и типы данных

В C каждая переменная имеет строгий тип.

| Тип      | Размер      | Диапазон                  |
|----------|-------------|---------------------------|
| `char`   | 1 байт      | -128..127                 |
| `int`    | 4 байта     | -2³¹..2³¹-1               |
| `float`  | 4 байта     | ~7 цифр точности          |
| `double` | 8 байт      | ~15 цифр точности          |

```c
int age = 20;
float pi = 3.1415;
char letter = 'A';
```
