# Урок 00: Введение в C

C — это компилируемый язык общего назначения, созданный в 1972 году
Деннисом Ритчи. Он лёг в основу UNIX и большинства современных ОС.

## Первая программа

```c
#include <stdio.h>

int main(void) {
    printf("Привет, Xodex!\n");
    return 0;
}
```

### Компиляция и запуск

```bash
gcc hello.c -o hello
./hello
```
