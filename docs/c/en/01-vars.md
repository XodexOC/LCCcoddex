# Lesson 01: Variables and types

Every variable in C has a strict type.

| Type     | Size     | Range                    |
|----------|----------|--------------------------|
| `char`   | 1 byte   | -128..127                |
| `int`    | 4 bytes  | -2³¹..2³¹-1              |
| `float`  | 4 bytes  | ~7 decimal digits        |
| `double` | 8 bytes  | ~15 decimal digits       |

```c
int age = 20;
float pi = 3.1415;
char letter = 'A';
```
