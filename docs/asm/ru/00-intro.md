# Урок 00: Введение в ассемблер

Ассемблер — это низкоуровневый язык, где каждая инструкция
соответствует одной машинной команде процессора.

## Инструменты

- `nasm` — ассемблер
- `ld` — компоновщик (linker)

### Сборка программы

```bash
nasm -f elf64 hello.asm -o hello.o
ld hello.o -o hello
./hello
```
