# Урок 01: Регистры и системные вызовы

## Основные регистры x86_64

| Регистр | Назначение                |
|---------|---------------------------|
| `rax`   | Аккумулятор / syscall №   |
| `rbx`   | Базовый регистр           |
| `rcx`   | Счётчик                   |
| `rdx`   | Регистр данных            |
| `rdi`   | Назначение (1-й аргумент) |
| `rsi`   | Источник (2-й аргумент)   |
| `rsp`   | Указатель стека           |
| `rbp`   | Указатель кадра стека     |

## Системные вызовы Linux

| Номер | Имя         | Описание          |
|-------|-------------|-------------------|
| 0     | `read`      | Чтение из файла   |
| 1     | `write`     | Запись в файл     |
| 60    | `exit`      | Завершение процесса|
