# Lab 04: Pointers / Указатели

**Track:** C Programming / Программирование на C  
**Slug:** `04-pointers`

## Task (EN)

Swap two ints via pointers. Explain why swap without pointers fails.

## Задание (RU)

Обмен двух int через указатели. Почему без указателей swap «не работает».
