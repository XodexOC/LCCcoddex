# Lab 10: Systems C (capstone) / Системный C (итог)

**Track:** C Programming / Программирование на C  
**Slug:** `10-systems-c`

## Task (EN)

Write a tiny shell loop: read line, fork, execvp, wait. Handle empty input.

## Задание (RU)

Мини-shell: read, fork, execvp, wait. Пустой ввод обрабатывать.

## Workflow

1. Read `docs/c/en/10-*.md` and `docs/c/ru/10-*.md`.
2. Implement your solution in this directory (`solution.*`).
3. Run `./test.sh`.
4. Mark the lesson complete in `~/.xodex/progress.json` when ready.

## Tools

Use what the Live image provides: `gcc`, `python3`, `rustc`/`cargo`, `sqlite3`, `man`, `valgrind`, etc.
