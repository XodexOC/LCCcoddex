# Lab 10: Systems thinking (capstone) / Системное мышление (итог)

**Track:** Linux Systems / Системы Linux  
**Slug:** `10-systems-thinking`

## Task (EN)

Map the Xodex boot path from ISO layout to login prompt in a short essay.

## Задание (RU)

Кратко опиши путь загрузки Xodex от ISO до login.

## Workflow

1. Read `docs/linux/en/10-*.md` and `docs/linux/ru/10-*.md`.
2. Implement your solution in this directory (`solution.*`).
3. Run `./test.sh`.
4. Mark the lesson complete in `~/.xodex/progress.json` when ready.

## Tools

Use what the Live image provides: `gcc`, `python3`, `rustc`/`cargo`, `sqlite3`, `man`, `valgrind`, etc.
