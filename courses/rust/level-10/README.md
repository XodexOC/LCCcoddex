# Lab 10: Systems Rust (capstone) / Системный Rust (итог)

**Track:** Rust / Rust  
**Slug:** `10-systems-rust`

## Task (EN)

Build a small CLI that processes a text file (manual args ok).

## Задание (RU)

CLI (ручные args) для обработки текстового файла.

## Workflow

1. Read `docs/rust/en/10-*.md` and `docs/rust/ru/10-*.md`.
2. Implement your solution in this directory (`solution.*`).
3. Run `./test.sh`.
4. Mark the lesson complete in `~/.xodex/progress.json` when ready.

## Tools

Use what the Live image provides: `gcc`, `python3`, `rustc`/`cargo`, `sqlite3`, `man`, `valgrind`, etc.
