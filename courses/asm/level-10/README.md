# Level 10 — C + ASM Capstone / Си + Ассемблер

**EN:** Write an assembly function `int square(int x)` that is callable from C.  
Then write a C program (`main.c`) that calls `square(7)` and prints the result.  
Compile both files with `gcc` (use `-no-pie`).

**RU:** Напишите на ассемблере функцию `int square(int x)`, вызываемую из C.  
Затем напишите C-программу (`main.c`), которая вызывает `square(7)` и печатает результат.  
Скомпилируйте оба файла с помощью `gcc` (используйте `-no-pie`).
