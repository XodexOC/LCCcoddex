# Level 05 — Addressing Modes / Режимы адресации

**EN:** Write a program that sums all elements of an array using indexed addressing (`[arr + rsi*4]`).  
Declare an array of 5 double-words: `10, 20, 30, 40, 50`. Use a loop and the `rsi` register as the index. Exit with the sum as the exit code.

**RU:** Напишите программу, которая суммирует все элементы массива, используя индексированную адресацию (`[arr + rsi*4]`).  
Объявите массив из 5 двойных слов: `10, 20, 30, 40, 50`. Используйте цикл и регистр `rsi` в качестве индекса. Завершите программу, передав сумму как код возврата.
