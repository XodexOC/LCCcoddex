# Level 09 — Strings / Строки

**EN:** Write a program that copies a string using `rep movsb`.  
Declare a source string `"Hello, World!\n"` and reserve space for a destination buffer.  
Use `cld` to clear the direction flag, set `rsi`, `rdi`, `rcx`, then execute `rep movsb`.  
Finally, print both the source and the copy.

**RU:** Напишите программу, которая копирует строку с помощью `rep movsb`.  
Объявите исходную строку `"Hello, World!\n"` и зарезервируйте место для буфера назначения.  
Используйте `cld` для очистки флага направления, установите `rsi`, `rdi`, `rcx`, затем выполните `rep movsb`.  
В конце выведите на экран исходную строку и её копию.
