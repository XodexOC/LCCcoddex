# Lab 00: Hello & Cargo / Hello и Cargo

**Track:** Rust / Rust  
**Slug:** `00-hello-cargo`

## Task (EN)

Create a binary crate and print Hello from Xodex.

## Задание (RU)

Создай binary crate и выведи Hello from Xodex.

## Workflow

1. Read `docs/rust/en/00-*.md` and `docs/rust/ru/00-*.md`.
2. Implement your solution in this directory (`solution.*`).
3. Run `./test.sh`.
4. Mark the lesson complete in `~/.xodex/progress.json` when ready.

## Tools

Use what the Live image provides: `gcc`, `python3`, `rustc`/`cargo`, `sqlite3`, `man`, `valgrind`, etc.
