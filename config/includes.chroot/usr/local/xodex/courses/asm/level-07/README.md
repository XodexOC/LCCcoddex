# Level 07 — Stack and Subroutines / Стек и подпрограммы

**EN:** Write a program with a `print_string` subroutine that prints a string using the `write` syscall.  
The subroutine should receive the string pointer in `rsi` and its length in `rdx`. Call it twice with two different messages.

**RU:** Напишите программу с подпрограммой `print_string`, которая печатает строку через системный вызов `write`.  
Подпрограмма должна получать указатель на строку в `rsi` и её длину в `rdx`. Вызовите её дважды с двумя разными сообщениями.
