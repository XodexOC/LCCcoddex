# Lab 00: Как работает компьютер / How computers work

**Track:** Assembly / Ассемблер
**Slug:** `00-how-computer-works`

## Task (EN)

This lab has NO code. Your task:
1. Draw a CPU with 4 registers (RAX, RBX, RCX, RDX) and RAM with addresses 0x00-0x0F.
2. Show with an arrow what `mov rax, [0x02]` does.
3. Convert to hex: 10, 31, 255. Convert to decimal: 0x20, 0xFF, 0x0A.
4. Explain in one sentence: what is a syscall?

## Задание (RU)

В этой лабораторной **нет кода**. Твои задачи:
1. Нарисуй CPU с 4 регистрами (RAX, RBX, RCX, RDX) и RAM с адресами 0x00-0x0F.
2. Стрелкой покажи, что делает `mov rax, [0x02]`.
3. Переведи в hex: 10, 31, 255. Переведи в десятичную: 0x20, 0xFF, 0x0A.
4. Объясни одним предложением: что такое syscall?
