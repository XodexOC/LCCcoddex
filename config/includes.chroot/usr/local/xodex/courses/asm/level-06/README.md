# Level 06 — Comparison and Jumps / Сравнение и переходы

**EN:** Write a program that finds the maximum of three numbers: `10, 25, 15`.  
Compare them using `cmp` and conditional jumps (`jge`, `jg`, etc.). Exit with the maximum value as the exit code.

**RU:** Напишите программу, которая находит максимум из трёх чисел: `10, 25, 15`.  
Сравните их с помощью `cmp` и условных переходов (`jge`, `jg` и т.д.). Завершите программу, передав максимальное значение как код возврата.
