# Level 08 — Calling Convention / Соглашение о вызовах

**EN:** Write a function `sum_three(a, b, c)` following the System V AMD64 ABI.  
The first three integer arguments are passed in `rdi`, `rsi`, `rdx`. Return the sum in `rax`.  
Call the function with values `10, 20, 30` and exit with the result.

**RU:** Напишите функцию `sum_three(a, b, c)` согласно соглашению о вызовах System V AMD64.  
Первые три целочисленных аргумента передаются в `rdi`, `rsi`, `rdx`. Верните сумму в `rax`.  
Вызовите функцию со значениями `10, 20, 30` и завершите программу с результатом.
