# Lab 09: Shell scripting / Shell-скрипты

**Track:** Linux Systems / Системы Linux  
**Slug:** `09-shell-scripting`

## Task (EN)

Write a script that backs up a directory with timestamp.

## Задание (RU)

Скрипт бэкапа каталога с timestamp.

## Workflow

1. Read `docs/linux/en/09-*.md` and `docs/linux/ru/09-*.md`.
2. Implement your solution in this directory (`solution.*`).
3. Run `./test.sh`.
4. Mark the lesson complete in `~/.xodex/progress.json` when ready.

Use what the Live image provides: `gcc`, `python3`, `rustc`/`cargo`, `sqlite3`, `man`, `valgrind`, etc.
