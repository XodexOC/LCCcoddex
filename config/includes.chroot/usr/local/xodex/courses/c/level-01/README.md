# Lab 01: Variables & types / Переменные и типы

**Track:** C Programming / Программирование на C  
**Slug:** `01-variables-types`

## Task (EN)

Print sizeof for int, long, size_t, char. Predict overflow of unsigned char.

## Задание (RU)

Выведи sizeof для int, long, size_t, char. Предскажи переполнение unsigned char.
