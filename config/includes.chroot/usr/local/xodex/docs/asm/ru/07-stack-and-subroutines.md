# [07] Стек и подпрограммы

> **Трек:** Assembly · **Уровень:** 07 · **Сложность:** ★★★☆☆

## 1. Суть

Чтобы не дублировать код, нужны **подпрограммы** (функции). Для их работы нужен **стек** — LIFO-структура в памяти, где хранятся адреса возврата, локальные переменные и сохранённые регистры.

## 2. Стек

- Растёт **вниз** (к младшим адресам)
- `RSP` — указатель вершины стека

```assembly
push rax    ; RSP -= 8; [RSP] = rax
pop  rax    ; rax = [RSP]; RSP += 8
```

## 3. call и ret

```assembly
call my_func   ; push адреса следующей инструкции; jmp my_func
ret            ; pop адрес; jmp по нему
```

## 4. Стековый фрейм

```assembly
my_func:
    push rbp          ; сохранить старый RBP
    mov  rbp, rsp     ; новый RBP = текущая вершина
    sub  rsp, 32      ; выделить 32 байта под локальные переменные

    ; [rbp - 8]  — первая локальная
    ; [rbp - 16] — вторая
    ; [rbp + 16] — аргументы (если на стеке)

    mov  rsp, rbp     ; восстановить RSP
    pop  rbp          ; восстановить RBP
    ret
```

## 5. Сохранение регистров

Неформальное правило:
- **Caller-saved:** rax, rcx, rdx, rsi, rdi, r8-r11 (функция может их менять)
- **Callee-saved:** rbx, rbp, r12-r15 (функция обязана сохранить)

```assembly
my_func:
    push rbx          ; сохраняем rbx
    push r12          ; сохраняем r12
    ; ... используем rbx и r12 ...
    pop  r12          ; восстанавливаем (обратный порядок)
    pop  rbx
    ret
```

## 6. Рекурсия: факториал

```assembly
factorial:
    push rbp
    mov  rbp, rsp
    cmp  rdi, 1
    jle  .base
    push rdi
    dec  rdi
    call factorial
    pop  rdi
    imul rax, rdi
    pop  rbp
    ret
.base:
    mov  rax, 1
    pop  rbp
    ret
```

## 7. Red Zone (Linux ABI)

128 байт под RSP, безопасные для временных данных в leaf-функциях (которые не вызывают другие функции).

## 8. Частые ошибки

- Несохранение callee-saved регистров (rbx, rbp, r12-r15)
- Дисбаланс стека: больше push, чем pop (или наоборот)
- Использовать `jmp` вместо `call` — ret некуда вернуться
- Забыть восстановить RSP после `sub rsp`
